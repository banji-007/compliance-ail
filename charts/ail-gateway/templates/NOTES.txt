── Required Next Steps ─────────────────────────────────────────────
1. Set your OpenAI API key (if not already set):
   helm upgrade {{ .Release.Name }} charts/ail-gateway \
     --set secrets.openaiApiKey=YOUR_OPENAI_KEY \
     -n {{ .Release.Namespace }}

2. Pull SPIRE sub-chart dependencies (first install only):
   helm dependency update charts/ail-gateway/

3. Build and push your custom images, then update values:
     images.controlPlane.repository / .tag
     images.dashboard.repository    / .tag
     images.agent.repository        / .tag

4. Expose the control plane and dashboard via Ingress:
   kubectl apply -f your-ingress.yaml -n {{ .Release.Namespace }}

   Then rebuild the dashboard image with the Ingress URL:
   docker build \
     --build-arg NEXT_PUBLIC_API_URL=https://your-ingress-url \
     --build-arg NEXT_PUBLIC_TENANT_ID={{ .Values.tenantId }} \
     -t <registry>/ail-dashboard:{{ .Chart.AppVersion }} ./dashboard
